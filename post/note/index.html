1 router-view用法

可以这样理解，正常写法中，一层路径(/xxx)对应一个router-view。

比如url: /，HelloWorld页面对应的就是App.vue中的router-view

2
3
4
5
6

routes: [
    {
      path: '/',
      component: HelloWorld
    }
  ]

2 scoped实现私有化样式的原理

查看DOM结构发现：vue通过在DOM结构以及css样式上加唯一不重复的标记，以保证唯一，达到样式私有化模块化的目的。

1

<h1 data-v-469af010>Welcome to Your Vue.js App</h1>
1
2
3

h1[data-v-469af010] {
    color: red;
}

如果组件内部包含有其他组件，只会给其他组件的最外层标签加上当前组件的data属性
